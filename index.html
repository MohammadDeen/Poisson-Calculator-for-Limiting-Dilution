Poisson Calculator for Limiting Dilution

Find the optimal lambda for limiting dilution cloning using Poisson statistics.

How to use this tool

Limiting dilution cloning distributes cells across wells so each well may receive 0, 1, or multiple cells. Poisson statistics describe this random dispensing process from the mean lambda value. Many labs target P(1) near 30-40% to balance monoclonal confidence against practical throughput. If lambda is too high, multi-cell contamination increases; if lambda is too low, too many wells are empty and plates are wasted.
